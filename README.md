A small application that shows Formula 1 seasons standings.  
  
### 🏠 [Homepage](https://f1-seasons.now.sh)
### 📄 [Documentation](https://github.com/ckomop0x/f1-seasons)

## Install

```sh
yarn install
```

## Usage

```sh
yarn start
```

## Run tests

```sh
yarn test
```

## Used technologies:
1. React
2. TypeScript
3. Styles Components
4. Webpack
5. Jest
